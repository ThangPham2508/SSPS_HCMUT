# SSPS_HCMUT
## Prerequisites
- NodeJS v18.13.0 or higher
- npm v8.19.3 or higher
#### Check your version with
```
node --version
npm --version
```

## Usage
#### Install Dependencies
```
npm install 
cd frontend
npm install
```
#### Run
- Make sure you are in /frontend folder.
```
npm run dev
```
